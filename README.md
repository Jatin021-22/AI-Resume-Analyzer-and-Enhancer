## 🤖 Overview

**AI Resume Analyzer & Enhancer** is a powerful AI-driven system designed to:

- Analyze resumes intelligently  
- Match candidates with job descriptions  
- Improve ATS (Applicant Tracking System) scores  
- Provide actionable insights for resume improvement  

> Built using modern NLP techniques and semantic embeddings to simulate real-world hiring systems.

---

## ✨ Features

| Feature | Description |
|--------|------------|
| 🧠 **Semantic Matching** | Matches resume with job description using embeddings |
| 📊 **ATS Score** | Calculates resume strength based on keywords & structure |
| 🧩 **Skill Clustering** | Groups skills using ML clustering (KMeans) |
| 🎯 **Role Detection** | Detects suitable job roles using similarity scoring |
| 📄 **PDF Parsing** | Extracts clean text from resumes |
| 📞 **Contact Extraction** | Detects email, phone, and key details |
| 🧱 **Section Detection** | Identifies sections like Education, Skills, Experience |
| ⚡ **Fast API** | High-performance backend using FastAPI |
| 🧪 **Tested System** | Includes unit tests for reliability |
